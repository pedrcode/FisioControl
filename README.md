# FisioControl

Sistema mobile para gerenciamento de atendimentos fisioterapêuticos, desenvolvido com foco em organização, praticidade e produtividade no dia a dia clínico.


 📱 Sobre o projeto

O **FisioControl** é um aplicativo que permite o controle completo de atendimentos, ajudando profissionais de fisioterapia a gerenciar pacientes, sessões e registros de forma simples e eficiente.

---

## 🚀 Funcionalidades

* Cadastro de pacientes
* Registro de atendimentos
* Edição de atendimentos
* Organização de histórico clínico
* Interface simples e intuitiva

---

## 🛠️ Tecnologias utilizadas

* React Native
* Expo
* JavaScript
* React Navigation

---

## ⚙️ Como rodar o projeto

### Pré-requisitos

* Node.js instalado
* Expo CLI instalado

```bash
npm install -g expo-cli
```

---

### Instalação

```bash
npm install
```

---

### Executando o projeto

```bash
npx expo start
```

Depois disso, você pode:

* Rodar no emulador Android
* Ou usar o app Expo Go no celular


## 📌 Objetivo

Este projeto foi desenvolvido com o objetivo de aprimorar habilidades em desenvolvimento mobile, além de resolver um problema real na organização de atendimentos fisioterapêuticos.
